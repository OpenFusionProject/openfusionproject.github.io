We're announcing the release of OpenFusionLauncher v2.1.3.

This release fixes a rare bug where the launcher could not resolve server IP addresses. It also now hides your password when using the password change dialog.

### Changelog

Full changelog available [here](https://github.com/OpenFusionProject/OpenFusionLauncher/compare/2.1.2...2.1.3).

## Download
The launcher can be grabbed from our [Downloads Page](https://openfusion.dev/download) or the [GitHub Release Page](https://github.com/OpenFusionProject/OpenFusionLauncher/releases/tag/2.1.3).
